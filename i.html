Судя по всему, вы пытаетесь отобразить список элементов с вкладками, используемыми для фильтрации. Есть много способов добиться этого; Я бы предложил использовать вычисленное свойство . Назовите переменную, filterкоторая представляет текущий фильтр (и текущую выбранную вкладку). Создайте вычисляемое свойство (назовем его courses), которое будет возвращать ваш список курсов для отображения в виде карточек (отфильтрованных на основе filterпеременной). new Vue({ el: '#tab', data: { filter: 'все', }, computed: { courses() { if (this.filter === 'все') { // Return all courses here } else { // Return filtered list of courses (based on this.filter) } } }, }) Поскольку coursesэто вычисляемое свойство, его значение будет обновляться при каждом filterизменении. Теперь в вашем HTML вы захотите использовать цикл (если вы еще этого не сделали) для отображения ваших курсов. Вам нужно будет перебрать вычисленное coursesсвойство, которое мы установили ранее.
